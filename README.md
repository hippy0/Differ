### About

Gendiff - it's a cli project for compares two configuration files with yml or json type.

### How to use

Setup:

> git clone git@github.com:hplskid/java-project-71.git
> 
> cd app
> 
> make build

Usage:

> cd app
> 
> make run-dist FILE1=(Path to file 1) FILE2=(Path to file two)
